API reference

Everything the platform exposes. Full interactive docs at /docs (Swagger).

GET/healthLiveness check with timestamp
GET/proto/{key}/runStream a live pipeline demo as Server-Sent Events (6 keys)
POST/chatRAG chat — question in, grounded answer + neural voice audio out
GET/greetPersonalized spoken greeting (base64 MP3)
GET/docsInteractive Swagger documentation for every endpoint
POST/api/auth/loginGet a JWT bearer token
GET/api/jobsScraped postings — filter by score, sponsorship, remote, freshness🔒
POST/api/jobs/scrapeTrigger a scrape of ~150 company boards now🔒
POST/api/jobs/{id}/trackMove a posting onto the application kanban🔒
POST/api/ai/cover-letterGenerate a tailored cover letter for a posting🔒
POST/api/ai/interview-prepTechnical + behavioral prep pack for a role🔒
GET/api/notificationsAlerts, briefings, follow-up reminders🔒
GET/api/health/jobsDiagnostics — config flags, posting counts, last cron runs🔒
